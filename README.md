### What I learned

- CSS property `aspect-ratio` to properly size svg images based on the original size, using just the height or width.
- CSS value `currentColor` for the svg fill property. 

```css
svg {
    aspect-ratio: 7 /1;
    fill: currentColor;
    height: 1.2rem;
}
```
